# Phase 8.2 Plan - Synthetic Data Generation

## Goal
Build a realistic generator for clients and documents to enable stress testing.

## Tasks
1. **Entity Factory**:
   - Create `src/lib/simulation/factory.ts`.
   - Implement `generateClients(count: number, seed?: string)` function.
   - Implement `generateDocuments(clientId: string)` with realistic status distributions.
2. **Scenario Templates**:
   - Define "Ideal Client", "Messy Client", and "High-Risk Client" profiles.
3. **Persistence Logic**:
   - Helper to flush generated entities to the database in batches.

## Tests to Implement
1. **Unit Test**: `tests/simulation/factory.determinism.test.ts`
   - Verify same seed produces identical client names and document types.
   - Verify different seeds produce varied distributions.
2. **Integration Test**: `tests/simulation/factory.db.test.ts`
   - Run generation and verify Prisma can retrieve the exact counts of records created.
   - Check document statuses match the expected distribution for "Messy Client" profile.
