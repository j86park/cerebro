# Plan 5.2: Design System & Layout Foundation

## Objective
Install shadcn/ui, configure Tailwind CSS v4, create global design tokens, and build the root layout with sidebar navigation.

## Context
- src/app/layout.tsx (currently bare)
- src/app/page.tsx (currently placeholder)
- SPEC: Stack is Tailwind CSS v4 + shadcn/ui + Recharts

## Tasks

<task type="auto">
  <name>Install shadcn/ui and Recharts</name>
  <action>
    Run `npx shadcn@latest init` and install required components.
    - Install: `badge`, `button`, `card`, `dialog`, `separator`, `skeleton`, `table`, `tabs`
    - Install Recharts: `npm install recharts`
    - Install Supabase client: `npm install @supabase/supabase-js`
  </action>
  <verify>npx tsc --noEmit</verify>
  <done>shadcn/ui components are available; Recharts and Supabase client installed.</done>
</task>

<task type="auto">
  <name>Configure global CSS design tokens</name>
  <files>src/app/globals.css</files>
  <action>
    Set up Tailwind CSS v4 design tokens for Cerebro's dark theme.
    - Background: deep charcoal/dark navy (not pure black)
    - Accent: electric blue (#3B82F6 family)
    - Status colors: red (EXPIRED/CRITICAL), amber (EXPIRING_SOON/HIGH), green (VALID/NONE)
    - Typography: Inter font (Google Fonts)
    - CSS custom properties: `--cerebro-bg`, `--cerebro-surface`, `--cerebro-border`, `--cerebro-accent`
  </action>
  <verify>Visual inspection on dev server</verify>
  <done>Design tokens are applied globally.</done>
</task>

<task type="auto">
  <name>Build root layout with sidebar</name>
  <files>src/app/layout.tsx, src/components/layout/Sidebar.tsx, src/components/layout/TopBar.tsx</files>
  <action>
    Replace bare layout with full app shell.
    - `layout.tsx`: Full-height flex with sidebar + main content area, Inter font import, globals.css import
    - `Sidebar.tsx`: Left sidebar with Cerebro wordmark, nav links (Dashboard, Vaults), queue status indicator
    - `TopBar.tsx`: Top bar with page title and DEMO_DATE badge
    - Layout is responsive: sidebar collapses to icon-only on narrow screens
  </action>
  <verify>npx tsc --noEmit && npm run dev (visual check)</verify>
  <done>App shell renders with sidebar and nav.</done>
</task>

<task type="auto">
  <name>Update home page to redirect to dashboard</name>
  <files>src/app/page.tsx</files>
  <action>
    Replace placeholder with `redirect('/dashboard')` using Next.js `redirect()`.
  </action>
  <verify>npx tsc --noEmit</verify>
  <done>Root page redirects to /dashboard.</done>
</task>

## Success Criteria
- [ ] shadcn/ui, Recharts, @supabase/supabase-js installed.
- [ ] Global dark theme with Cerebro palette.
- [ ] App shell with sidebar renders on `npm run dev`.
- [ ] Root page redirects to `/dashboard`.
