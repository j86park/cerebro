# Plan 5.5: Supabase Realtime Integration

## Objective
Implement the Supabase Realtime client for live dashboard updates. When an agent logs an action, the dashboard reflects it without a page refresh.

## Context
- docs/architecture.md §7 (Realtime Layer)
- Channel 1: `cerebro-agent-actions` → INSERT on agent_actions
- Channel 2: Dashboard vault grid urgency refresh on new actions
- @supabase/supabase-js (installed in Plan 5.2)

<task type="auto">
  <name>Create useAgentActions realtime hook</name>
  <files>src/hooks/useAgentActions.ts</files>
  <action>
    React hook that subscribes to new agent actions for a specific client.
    - Takes `clientId` as param
    - Subscribes to Supabase Realtime channel `cerebro-agent-actions`
    - Filters INSERT events where `client_id = clientId`
    - Returns `{ actions, isConnected }` state
    - Cleans up subscription on `clientId` change or unmount
  </action>
  <verify>npx tsc --noEmit</verify>
  <done>Hook subscribes, receives inserts, and cleans up correctly.</done>
</task>
